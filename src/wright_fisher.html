Wright-Fisher Model for genetic Drift.

As described in class the Wright-Fisher model is an analytical treatment to understand the effect of drift on population dynamics. We emphasized a lot that after having taken this class we want you to think about evolution in a different way in which deterministic Darwanian forces of selection are not the only evolutionary driving force. Many of the features seen today in many organisms, including humans, are a reflect of small sampling number and genetic drift rather than traits shaped by natural selection.

Now that we know how to build the matrix let's just get to it right away!

First we need to define the population size as well as initialize the matrix.

% define population size.

N = 16; % as in Burin's experiment

% initialize transition matrix

% the dimensions of the matrix are 2N+1 because we need to include the zero.

P = zeros(2 * N + 1, 2 * N + 1);

Now we use a pair of nested for loops to fill all the entries of the matrix.

% use for loops to build the matrix

for i=0:(2 * N)

for j=0:(2 * N)

P(j + 1, i + 1) = nchoosek(2 * N, j) *...

(i / (2 * N))^j *...

(1 - i / (2 * N))^(2 * N - j);

end

end

Once the matrix is ready we can apply it to our vector.

We will initialize a matrix X of n_gen where on each column we'll keep track of each generation. For the first generation we will use Burin's starting conditions where we have 16 heterozygous flies, i.e. 16 alleles of type 1 or frequency = 1.

% define number of generations to run the simulation

n_gen = 20;

% initialize vector for allele frequency

X = zeros(2 * N + 1, n_gen);

% initilize the value for the first generation where we

% have 16 copies of allele 1.

% We modify entry 17 because we start counting at zero!

X(N + 1, 1) = 1.0;

Now we can just loop through the number of generations and compute the probabilities of each allele copy number.

% apply the matrix on each generation

for k=2:n_gen

X(:, k) = P * X(:, k - 1);

end
