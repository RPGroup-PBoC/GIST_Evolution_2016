Lesson 1: The Hello, world program.

These tutorials is based on Justin Bois' excellent programming bootcamp. The language of selection for our course is different though. We will be using the popular numerical computing environment MATLAB. But the logic behind every programming language is almost identical.

Traditionally, the first program anyone writes when learning a new language is called "Hello, world." In this program, the words "Hello, world." are printed on the screen. The original Hello, world. was likely written by Brian Kernighan, one of the inventors of Linux, and the author of the classic and authoritative book on the C programming language. In his original, the printed text was "hello, world" (no period or capital H), but people use lots of variants.

In these lessons, which are generated using Matlab's new live scripts code is shown and exectuted in special cells, like the one below.

% This is a code cell.

% It does not contain any executable code, just comments.

In looking at this code cell, we have already learned our first bit of Matlab's syntax. Lines beginning with the pound sign (%) are comments and are ignored.

Ok, now a code cell with Hello, world.

'Hello, world'

ans = Hello, world

Hooray! We just printed Hello, world. to the screen. To do this, we just wrote a command on the terminal. If a command does not have a semicolon (;) at the end then it is printed on the screen. Let's look at an example

'This will not be printed on the screen because of the semicolon';

'This will apear on the screen'

ans = This will apear on the screen

With this in hand and our ability to make the Matlab interpreter greet the world, we are now ready to venture into programming!
